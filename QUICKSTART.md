# 🚀 Quick Start Guide

## First Time Setup

### 1. Create Your PIN
When you first open the app:
- Enter a 4-digit PIN (remember this!)
- Confirm your PIN
- 4 sample habits are automatically created for you

### 2. Start Tracking
- Check off habits as you complete them daily
- See your progress percentage at the top
- Build streaks! 🔥

## Daily Use

### Main Screen (Habits)
- ✅ **Check boxes** - Mark habits complete
- **Streaks** - See your consecutive days
- **Navigation** - Swipe between days
- **Add button** (+) - Create new habits

### Calendar View
- **Heatmap** - Visual completion overview
- **Colors** - Show completion percentage
- **Perfect days** - 100% completion marked green
- **Monthly stats** - See total check-ins

### Statistics
- **Current streak** - Days in a row
- **Best streak** - Your record
- **Graphs** - Weekly and monthly trends
- **Per-habit stats** - Individual performance

## Managing Habits

### Add New Habit
1. Tap the **+** button
2. Enter habit name (e.g., "Yoga")
3. Add description (optional)
4. Choose a color
5. Tap "Add Habit"

### Delete Habit
1. Tap **⋮** (three dots) on main screen
2. Select "Manage Habits"
3. Tap trash icon next to habit
4. Confirm deletion

## Data Management

### Export Data (Backup)
Settings → Export Data → Save JSON file

**Do this regularly!** It's your backup.

### Import Data (Restore)
Settings → Import Data → Select JSON file

Use this to:
- Restore from backup
- Move to new device
- Transfer between browsers

### Clear All Data
Settings → Clear All Data → Confirm

⚠️ **Warning**: This deletes everything! Export first.

## Troubleshooting

### Forgot PIN?
Unfortunately, you'll need to:
1. Clear browser data for this site
2. Start fresh with new PIN
3. *That's why exporting data regularly is important!*

### Data Disappeared?
- Check if you're in incognito/private mode (doesn't save)
- Browser data might have been cleared
- Restore from your last export

### App Not Working Offline?
- First visit requires internet
- After that, works 100% offline
- Service worker caches everything

### Charts Not Showing?
- Need at least 2 days of data
- Check that you're completing habits
- Calendar needs completed habits to display

## FAQ

**Q: Does this need internet?**
A: No! Works 100% offline after first load.

**Q: Where is my data stored?**
A: In your browser's IndexedDB. Private and local.

**Q: Can I use on multiple devices?**
A: Yes! Export from one, import to another.

**Q: Is my data private?**
A: Absolutely. Never leaves your device.

**Q: Can I sync between devices?**
A: Not automatically. Use export/import.

**Q: What if I clear browser data?**
A: Data is lost unless you have an export.

**Q: How many habits can I track?**
A: Unlimited! But 5-10 is most effective.

**Q: Can I change my PIN?**
A: Feature coming soon. For now, export data and start fresh.

## Support

Need help?
1. Check browser console (F12) for errors
2. Try clearing cache and reloading
3. Export data before troubleshooting
4. Last resort: clear site data and reimport
